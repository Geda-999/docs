# 日本漫画

这里只是代表网站以日本漫画为主打，不代表没有其他国家漫画。

  • 腾讯动漫： https://ac.qq.com/ 国产原创漫画很多，小说改编漫画也不少，人气高的还有机会动画化。有APP (opens new window)
