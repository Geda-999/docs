# qiandao

GitHub (opens new window) qiandao是一个python的自动签到框架，开源项目。可以安装在服务器、VPS、群晖等，安装完成后添加签到模板就可以开始自动签到了。官网好像是2014年开始运营，安全性方面应该问题不大

# 签到站网友自建服务

第三方服务，毕竟会把账号密码或cookie提交到别人的服务器，自行考虑不做推荐，有能力建议自己建服务器。只要网友的站没关闭注册都可以蹭，网站上没有提供的模板可以到 官方网站》点击“查看所有公开模板” https://qiandao.binux.me/tpls/public (opens new window) 》点击“查看”》点击下方的“下载”，下载到的har文件上传到你要使用的网站即可

另外需要注意，有些网站可能会屏蔽外国IP，所以使用别人的签到站或自己买服务器架设签到站需要考虑，是买国内服务器还是国外服务器，使用国外服务器签到使用的就是国外IP。

# 魂签

官方GitHub (opens new window)，一个chrome的插件，可以自动签到，每天只要打开浏览器就可以自动签到，不用填写账号密码，不用手动抓取cookie，只需要 添加脚本+登录账号。已经支持火狐浏览器

使用方法： 手动安装: 点击下载插件 (opens new window)，拖动到扩展程序中 chrome网上应用店安装 (opens new window) firefox安装 (opens new window)

添加安装脚本： 最简单的方法，点击插件图标->任务管理->脚本推荐->挑选安装 第三方或者自用脚本安装方法, 点击插件图标->任务管理->添加脚本->上传脚本文件/粘贴脚本代码/拖拽脚本文件/粘贴脚本链接

魂签脚本发布站点：https://soulsign.inu1255.cn/ (opens new window)
